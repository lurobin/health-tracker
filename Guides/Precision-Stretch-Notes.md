# 精准拉伸 — 参考笔记

> 来源：Kristian Berg《精准拉伸：疼痛消除和损伤预防的针对性练习》
> 提取日期：2026-05-14
> 重点：膝盖康复期可用的拉伸方案

---

## 一、拉伸核心原则（PNF 法）

### 4大基本原则
1. **避免疼痛** — 疼痛触发肌肉防御性收缩，与拉伸目的相反
2. **缓慢拉伸** — 快速拉伸触发牵张反射，肌肉反而收紧
3. **拉伸正确的肌肉** — 方向错误 = 拉关节囊而非肌肉
4. **避免影响其他肌肉和关节** — 姿势不规范会让状况更糟

### 黄金准则
> 以最小的关节活动达到最大限度的拉伸

### PNF 法 6 步骤
1. 设定正确的初始姿势
2. 拉伸至终止点（轻微刺痛感）
3. 放松
4. 在不动肌肉的前提下收紧肌肉（对抗阻力 5-10 秒）
5. 放松
6. 拉伸至新的终止点
重复 3-6 次

## 二、绝对禁止的拉伸动作

| 禁止动作 | 原因 |
|:---|:---|
| 站姿拉伸大腿后侧 | 弓背 + 膝关节压力 |
| 站姿拉伸大腿内侧（劈叉） | 大腿内侧压力过大 |
| 卧姿大腿前侧（脚跟贴臀） | 腰椎过度活动 + 膝关节极限屈曲 |
| 跪姿拉伸大腿前侧 | 腰椎弯曲 + 膝关节极限屈曲 |
| 站姿拉伸大腿前侧 | 腰椎主导 + 膝关节极限屈曲 |
| 站姿拉伸髋部屈肌（腿后伸） | 增加腰椎弯曲弧度 |
| 坐姿拉伸臀肌（圆背） | 腰部应呈弓形而非圆形 |

## 三、与膝盖/腰部直接相关的肌肉

### 1. 梨状肌（Piriformis）★★★ 每天必拉伸
- **位置**：臀大肌深层，骶骨→股骨大转子
- **紧绷后果**：臀部痛、大腿后侧→膝后侧放射痛、膝盖外侧痛（跑步膝）、腰痛
- **紧绷原因**：久坐、双脚分开宽距坐立
- **站姿拉伸**：腹股沟同高平面，膝呈直角，骨盆和腿形成开放方形，上半身前倾保持腰背拱度
- **坐姿拉伸**：脚踝放对侧膝上，下压膝盖或上半身前倾
- **警告**：膝盖内侧/外侧痛或腹股沟不适时不做

### 2. 髋部屈肌（髂腰肌）★★★
- **位置**：深层，下腰椎/髋骨前侧→股骨上端内侧
- **紧绷后果**：腰背痛、腹股沟/大腿内侧痛
- **紧绷原因**：久坐（髋部持续收缩）
- **安全拉伸**：平躺桌边，一腿抱胸贴桌面，另一腿自然悬空下垂
- **注意**：大多数站姿髋部屈肌拉伸是错的（增加腰椎弧度）

### 3. 股直肌（Rectus Femoris）★★★ 唯一跨膝+髋的股四头肌
- **位置**：臀部前端→髋关节和膝关节→髌腱
- **紧绷后果**：腰背痛、**膝盖骨周围疼痛**
- **紧绷原因**：久坐、跑步、足球、骑行
- **安全拉伸（卧姿）**：俯卧长凳，绳套脚，手臂过头拉绳，保持腰背不拱
- **跪姿版**：背对墙，一脚前踩，另一膝向后滑近墙，膝弯90°
- **警告**：膝盖骨周围有伤病时不练

### 4. 阔筋膜张肌（TFL / IT Band）★★
- **位置**：臀部外侧前部→胫韧带（IT Band）→膝外侧→胫骨上端
- **紧绷后果**：臀部弹响髋、大腿外侧痛、**膝盖外侧痛/跑步膝**、膝盖骨周围痛
- **紧绷原因**：久坐、跑步、攀岩、骑行
- **拉伸**：与股直肌跪姿类似，但上半身和腿形成弓形

### 5. 股后肌群（腘绳肌）★★
- **功能**：收缩膝关节、伸展臀部
- **紧绷后果**：腰背痛、步幅缩小、大腿后侧痉挛风险增加
- **安全拉伸**：坐凳面，腿伸直（脚在凳边缘外），上半身前倾保持腰背拱度
- **警告**：膝盖骨周围痛或跟腱只有拉伸感时不做

### 6. 腓肠肌（Gastrocnemius）★
- **结构**：两个头起于股骨下端后侧，形成跟腱→脚后跟
- **功能**：绷脚尖 + **弯曲膝盖**（跨膝关节）
- **特点**：需要较长时间/较大力量才达到真正拉伸效果

### 7. 比目鱼肌（Soleus）★
- **结构**：腓肠肌下层，**不跨膝关节**—与腓肠肌的关键区别
- **功能**：仅绷脚尖
- **安全提示**：比目鱼肌紧绷也会导致跟腱问题
- **拉伸要点**：保持膝盖弯曲（防止动用腓肠肌）

## 四、何时避免拉伸

| 绝对禁忌 | 需要谨慎 |
|:---|:---|
| 骨折后 | 肌肉拉伤后 48 小时内 |
| 高烧时 | 关节扭伤（需先诊断） |
| 关节发炎时 | 可的松注射后 10 天内 |
| 皮肤有开放性创面 | 关节过度活动症 |

## 五、与用户当前状况的关联

### 膝盖康复期优先拉伸顺序
1. **梨状肌** — 每天，坐姿版优先（安全）
2. **髋部屈肌** — 每天，桌边卧姿版
3. **股直肌** — 隔天，卧姿版优先（减少腰椎参与）
4. **阔筋膜张肌** — 隔天，注意膝盖不适即停
5. **腓肠肌+比目鱼肌** — 散步/有氧后

### 需避免的动作
- 所有站姿大腿前侧/后侧/内侧拉伸
- 任何导致膝盖极限屈曲的动作
- 任何引起膝盖内侧刺痛的动作

### 可用工具
- 墙壁、桌椅、毛巾、绳子（围巾/腰带替代）
- 全部主动拉伸，不需他人辅助
